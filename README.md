# 📊 Customer Shopping Insights Dashboard

## 🔹 Overview

This project presents an interactive Power BI dashboard designed to analyze customer shopping behavior, spending patterns, and business trends.

The goal of this dashboard is to transform raw data into meaningful insights that help understand customer preferences, revenue drivers, and overall performance.

-----

## 🔹 Dataset

* Source: Kaggle
* The dataset includes:

  * Customer ID
  * Age & Gender
  * Product Category
  * Purchase Amount
  * Payment Method
  * Subscription Status
  * Ratings & Purchase History

------

## 🔹 Tools Used

* Power BI – Dashboard creation and interactive visualization
* DAX – Creating calculated measures and KPIs
* Excel – Data cleaning and preprocessing
* SQL – Data querying and analysis

------

## 🔹 Key KPIs

* Total Revenue
* Total Customers
* Average Rating
* Revenue per Customer
* High-Value Customers
* Repeat Customers

------

## 🔹 Dashboard Features

### 📌 Revenue Analysis

* Revenue by Product Category
* Revenue by Age Group
* Revenue by Season

### 📌 Customer Insights

* Customer Distribution by Gender
* Subscription Status Analysis
* Repeat vs High-Value Customers

### 📌 Payment Insights

* Payment Method Distribution

### 💡 Key Insight

* Identifies top-performing customer segment dynamically using DAX
